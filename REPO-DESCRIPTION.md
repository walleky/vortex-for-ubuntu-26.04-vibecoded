# vortex for ubuntu 26.04 -vibecoded-

Open-source Linux helper for running Nexus Mods Vortex through Steam Proton on Ubuntu 26.04, focused on Skyrim Special Edition.

It handles:

- Vortex install through Proton
- Skyrim SE Proton prefix detection
- SKSE64 install and launch
- `nxm://` browser links
- optional Nexus Mods API-assisted downloads
- Nexus Collections handoff to Vortex
- Non-Nexus archive imports
